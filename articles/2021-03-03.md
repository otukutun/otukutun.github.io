## データ構造とアルゴリズム day12
今回は[この問題](https://leetcode.com/problems/is-subsequence/)をやりました。さくっと解けた。tの文字れ数をNとすると、時間計算量はO(N), 空間計算量はO(1)となる。

```ruby
# @param {String} s
# @param {String} t
# @return {Boolean}
def is_subsequence(s, t)
  sub_i = 0
  t.each_char do |c|
    sub_i += 1 if c == s[sub_i]
  end
  # 最終的にsubsequenceの場合、sub_iは最大値+1になる
  sub_i == s.length
end
```
